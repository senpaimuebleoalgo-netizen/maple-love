# 🍁 MapleStory Weekend Party Invitation 🍁

A fun, interactive web-based invitation for a MapleStory-themed party weekend!

## Features ✨

- **Interactive UI** - MapleStory-styled game window interface
- **Falling Maple Leaves** - Continuous animation of falling leaves
- **Floating Hearts** - Heart particles that appear when accepting the invitation
- **Background Music** - Play/pause Ellinia background music
- **Countdown Timer** - Displays days, hours, and minutes until the party
- **Runaway "No" Button** - The "No" button runs away when you hover over it 😈
- **Yes/No Interaction** - Click "Yes" to create the party with celebration effects

## How to Use

1. Open `index.html` in your web browser
2. Click the music button to play the background music
3. Read the invitation from Cholo to Crissy
4. Click "Sí ❤️" to accept the party invitation (celebration effects!)
5. Or try to click "No 😈" - but it won't let you! 😄

## Technologies Used

- HTML5
- CSS3 (Animations & Keyframes)
- Vanilla JavaScript (No frameworks needed!)

## Customization

- Edit character names and levels in `index.html`
- Modify animations in `style.css`
- Adjust countdown timer date in `script.js`
- Replace assets with your own images and music
